推荐·实用的Chrome插件们

发表于 Sun, Jun 16th, 2019 | | 阅读次数:
字数统计: 910 | 阅读时长 ≈ 3

前言

Chrome有很多实用的插件，安装它们可以大大提高效率，本文列举了一些常用的插件。

不过Chrome作为内存杀手，插件一多内存分分钟飙满QwQ

安装

首先，安装插件是最重要的。

如果有能力，可以到Chrome网上应用店下载安装插件，安全方便快捷。

但是众所周知，由于某些原因，国内无法直接访问Google，所以可以在Crx4Chrome等网站下载.crx格式的Chrome插件，国内可以正常访问。

获得了.crx文件后，如何安装呢？自Chrome 67以后，就不能通过离线安装的方式安装本地插件了。因此，我们需要将后缀名改为.zip然后解压，进入chrome://extensions/扩展程序管理页面，打开开发者模式，单击加载已解压的扩展程序，选择解压出的文件夹即可。

插件

Adblock Plus

Adblock Plus广受好评，是使用人数最多的广告拦截插件之一。它可以拦截包括横幅、弹出窗口、YouTube视频广告在内的绝大多数广告，同时拥有Android和iOS版本。ABP还提供了丰富的自定义功能，甚至可以用来通过屏蔽部分功能加快某些网站的访问速度。（例如CurseForge）

Infinity新标签页

Infinity新标签页是一款美观、自定义程度高且实用的新标签页。它拥有丰富的壁纸库，支持云端备份。

Chrome商店地址
官网地址

广告终结者

广告终结者与前面介绍过的ABP类似，同样可以拦截广告。但广告终结者主要侧重于视频广告，例如ABP拦截不了爱奇艺广告，但广告终结者可以完美去广告。

Chrome商店地址

Chrono下载管理器

Chrome内置的下载管理器十分鸡肋，功能少之又少。而Chrono完美地解决了这个痛点，添加了许多实用的功能，包括但不限于新建任务、回收站、文件预览、通知栏提醒等等，并且Chrono能够无缝嵌在Chrome中。



Chrome商店地址
官网地址

OneTab

众所周知，Chrome的内存优化极其智障，如图，仅仅打开了一个标签页，内存占用就如此之高（当然与启用的扩展数量也有关系）。

如果使用OneTab休眠标签页，就能减少不少内存占用。根据官方的描述，OneTab可以

节省高达95％的内存，并减轻标签页混乱现象

可以看到，内存占用从原本的1098MB降到761MB，效果明显。

IE Tab

IE Tab能以IE内核打开网页，功能略显鸡肋，毕竟现在绝大多数网站都支持Chromium内核。

Imagus

Imagus是一个非常好用的图片预览插件，鼠标悬停在图片上即可预览大图（虽然有时候会误操作很烦）。

Chrome商店地址

SimpleUndoClose

这个插件添加了一个查看最近标签页的按钮，可以很方便地撤销对标签页的误删除，以后再也不用Ctrl+H查看历史记录啦！（什么？Ctrl+Shift+T？抱歉，这个快捷键一次只能撤销一个标签页）

Tampermonkey

至于油猴就不用说了，万能插件，脚本多的是，功能多到爆炸。不仅支持Chrome，还支持Edge、Safari、Firefox等等。
