链表

发表于 Wed, Jul 31st, 2019 | | 阅读次数:
字数统计: 1.6k | 阅读时长 ≈ 8

简介

链表是一种线性数据结构，最大的特点就是存储不连续，可以用于实现邻接表等。

链表分为单链表、双向链表和循环链表三种，如下图所示：


可以看出，单链表中每个节点都有一个next指针，指向后$1$个节点，而最后$1$个节点的next为空指针；
双向链表的每个节点还有一个prev指针，指向前$1$个节点；
循环链表则是在上面的基础上让最后$1$个节点的next指向头节点，如果是双向循环链表还会让头节点的prev指针指向最后$1$个节点。

LinkedList的方法有$10$个：

  • getHead()：返回head指针
  • getEnd()：返回最后$1$个节点的指针
  • locate(int pos)：返回第pos个节点的指针
  • ins(type data)：将data插入到末尾
  • ins(type data, int pos)：重载函数，将data插入到pos的位置
  • del()：删除末尾节点
  • del(int pos)：重载函数，删除第pos个节点
  • size()：返回节点数
  • print()：遍历链表，输出每个节点的值
  • empty()：返回链表是否为空
